Element Selection Test Page

Test the Chrome extension's element selection functionality

How to Test Element Selection:

  • Keyboard Shortcut: Press Ctrl+Shift+S to start element selection
  • Extension Button: Click the extension icon and use the element selection feature
  • Hover: Move your mouse over elements to see them highlighted
  • Select: Click on an element to select it
  • Cancel: Press ESC or right-click to cancel selection
